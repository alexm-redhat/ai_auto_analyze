FUSION NOTES
------------
With fuse_allreduce_rms=true, the torch.compile pass fuses:
  (a) RowParallelLinear allreduce + RMSNorm + residual_add → single TRT-LLM fused kernel
      Applies at: o_proj allreduce → post_attn_layernorm
      Applies at: dense MLP down_proj allreduce → next layer's input_layernorm
      Applies at: MoE allreduce → next layer's input_layernorm
  (b) Potentially: allreduce + RMSNorm + residual_add + NVFP4 quantization → single kernel
      Applies when the fused norm output feeds directly into an NVFP4 GEMM.

The attention op (vllm::unified_mla_attention_with_output) is a graph-splitting op,
meaning the compiled graph is split at each attention call. This produces two compiled
subgraphs per layer: pre-attention and post-attention segments.

================================================================================
CROSS-LAYER BOUNDARY EFFECTS
================================================================================

1. Layer N MoE output → AllReduce (op 24) → Layer N+1 Input RMSNorm (op 1)
   With fuse_allreduce_rms=true, the MoE AllReduce and the next layer's
   Input RMSNorm + Residual Add are fused into a single TRT-LLM kernel:
   trtllm_fused_allreduce_norm (pattern: kARResidualRMSNorm).

2. Layer N o_proj output → AllReduce (part of op 15) → Post-Attn RMSNorm (op 16)
   Similarly fused into trtllm_fused_allreduce_norm. The RowParallelLinear
   GEMM output feeds into allreduce which is fused with the post-attention
   RMSNorm + residual add.

3. CUDA Graph Splitting: The attention operation (vllm::unified_mla_attention_with_output)
   is a graph-splitting op. The compiled graph is split into two segments per layer:
   - Pre-attention segment: input_layernorm → QKV projections → RoPE → KV cache store
   - Post-attention segment: output projection → post_attn_layernorm → MLP/MoE
   Each segment is independently compiled and captured in CUDA graphs.

4. Compile Ranges: Two compile ranges are used:
   - Range (1, 73): Small batch sizes (decode-dominant)
   - Range (74, 8192): Larger batch sizes (prefill-dominant)

5. The last layer (layer 60) output goes through a final RMSNorm (not fused with
   allreduce since it's the terminal norm) followed by the LM head projection.
